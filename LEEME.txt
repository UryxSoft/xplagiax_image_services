CAMBIOS_.TAR.GZ — cambios de esta sesión
==========================================

CONTEXTO IMPORTANTE
--------------------
El PR de la rama anterior (claude/reverse-image-search-microservice-eq9ti9)
ya fue mergeado a `main` (commits 41e4fa8 y de903ca). Ese merge incluía la
primera versión del microservicio de reverse image search + el endpoint de
batch. Este tarball contiene SOLO el trabajo NUEVO hecho después de ese
merge, aplicado sobre el `main` actual:

  1. Mejoras de velocidad y exactitud:
     - REVERSE_SEARCH_REQUEST_DEADLINE_S (presupuesto de tiempo por request)
     - REVERSE_SEARCH_MODE=latency (llamar a todos los proveedores en
       paralelo, opt-in; por defecto sigue siendo "cost"/Early Stop)
     - REVERSE_SEARCH_BATCH_CONCURRENCY (pool acotado para /batch)
     - Similarity graduada + corroboración entre proveedores + bonus/penalty
       de dominios de confianza (todo opt-in, vacío por defecto)
  2. Circuit breaker por proveedor (reusa app/services/circuit_breaker.py,
     Redis, misma clase que ya usa api_rotator.py) para fallos SOSTENIDOS
     entre requests, no solo dentro de un mismo request.
  3. El deploy standalone (app.reverse_search.app) ahora también sirve
     /api/v1/patents/* (SerpApi/Zenserp vía api_rotator.py + patents_bp),
     usando un adaptador nuevo (_TempHostImageStorage) para reusar el
     hosting efímero ya existente. Sigue sin servir /api/v1/images ni
     /api/v1/search/* (necesitan CLIP/SigLIP/Qdrant).
  4. Métricas Prometheus específicas del endpoint de reverse-image-search
     (antes solo se reusaban las HTTP genéricas).

`patent_image_search`, `api_rotator.py` y `patents_bp` en el monolito
(app/factory.py) NO fueron tocados — cero cambios ahí, verificado con
`git diff`.

CÓMO APLICAR
------------
Opción A (recomendada) — aplicar el patch sobre un `main` limpio:

    git checkout main
    git pull
    git checkout -b claude/reverse-image-search-microservice-eq9ti9
    git apply CAMBIOS.patch
    git add -A
    git commit -m "Add speed/accuracy tuning, circuit breaker, patents in standalone"
    git push -u origin claude/reverse-image-search-microservice-eq9ti9

Opción B — copiar los archivos sueltos de este tarball directamente sobre tu
copia local del repo (mismas rutas relativas), y luego commitear a mano.
